3.1 IOжөҒеҺҹзҗҶ

IOжөҒпјҲInput/Output Streamпјүпјҡ з”ЁдәҺеӨ„зҗҶж•°жҚ®дј иҫ“гҖӮ

еҺҹзҗҶпјҡ

иҫ“еҮәпјҡзЁӢеәҸпјҲеҶ…еӯҳпјү вҶ’ OutputStream/Writer вҶ’ зӣ®зҡ„ең°пјҲж–Үд»¶пјү

зү№зӮ№пјҡ

  • рҹ“– IOжөҒд»ҘжөҒзҡ„ж–№ејҸиҝӣиЎҢж•°жҚ®дј иҫ“
  • рҹ“– еҚ•еҗ‘дј иҫ“пјҡиҫ“е…ҘжөҒеҸӘиҜ»пјҢиҫ“еҮәжөҒеҸӘеҶҷ
  • рҹ“– е…Ҳиҝӣе…ҲеҮәеҺҹеҲҷ

3.2 жөҒзҡ„дҪ“зі»з»“жһ„

Java IOжөҒеӣӣеӨ§жҠҪиұЎеҹәзұ»пјҡ

зұ»еһӢеӯ—иҠӮжөҒеӯ—з¬ҰжөҒ
иҫ“е…ҘжөҒInputStreamReader
иҫ“еҮәжөҒOutputStreamWriter

IOжөҒдҪ“зі»еӣҫпјҡ

еӯ—иҠӮжөҒпјҡ
в”ңв”Җ InputStream (жҠҪиұЎзұ»)
в”Ӯ  в”ңв”Җ FileInputStream        (ж–Үд»¶)

в”Ӯ  в””в”Җ ObjectInputStream      (еҜ№иұЎ)
в””в”Җ OutputStream (жҠҪиұЎзұ»)
   в”ңв”Җ FileOutputStream       (ж–Үд»¶)
   в”ңв”Җ BufferedOutputStream   (зј“еҶІ)
   в””в”Җ ObjectOutputStream     (еҜ№иұЎ)

еӯ—з¬ҰжөҒпјҡ
в”ңв”Җ Reader (жҠҪиұЎзұ»)
в”Ӯ  в”ңв”Җ FileReader            (ж–Үд»¶)

   в”ңв”Җ BufferedWriter        (зј“еҶІ)
